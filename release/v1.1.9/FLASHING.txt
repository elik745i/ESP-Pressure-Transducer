ESP Pressure Transducer v1.1.9

Flash offsets for ESP8266 4M1M / Wemos D1 mini:
- firmware.bin -> 0x000000
- littlefs.bin -> 0x300000

Notes:
- Flash firmware.bin for normal firmware updates.
- Flash littlefs.bin too only when you intentionally want to refresh the filesystem image.
- Wi-Fi and MQTT credentials are stored in persistent EEPROM-backed storage.
- Added retained buzzer volume control in the web UI and over MQTT.
- Added retained touch sensor enable / disable in the web UI and over MQTT.
- Improved alarm and touch handling to reduce unwanted beeps and report low/high alarm state separately.
